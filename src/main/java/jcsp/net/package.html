This is main package for JCSP.NET.

This package is now deprecated and should no longer be used; the package jcsp.net2 repalces this package.

Before any JCSP.NET constructs can be used, the JCSP Node must be initialized. See the {@link jcsp.net.Node} class for more information.

Networked channels can be constructed using the {@link jcsp.net.NetChannelEnd} class or by using {@link jcsp.net.StandardNetChannelEndFactory}.

See the {@link jcsp.net.cns} package for information on using named channels.
